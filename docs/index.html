Not a chatbot. Not an app.

How it works

01  /  Install

Download and run in under two minutes

A native installer for Mac or Windows. No dependencies, no configuration, no dev environment to set up. openall runs entirely in your environment — nothing leaves without your say.

02  /  Connect

Bring your own LLM

Paste in your API key for the LLM provider of your choice. openall connects directly to the model. No accounts, no subscriptions, no middleman. Your key, your model, your control.

03  /  Describe

Say what you need, want, or imagine

No syntax. No boilerplate. No scaffolding. Describe your intent in plain language. A tool. A workflow. A data transformation. A product. Anything.

04  /  Done

The LLM generates and runs it

Your chosen model generates, executes, and evolves the result in real time. No code is written. No file is saved. The model is the application.
